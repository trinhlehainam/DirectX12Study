■MMD用モデルデータ　博麗霊夢（ Reimu Hakurei ）

******* 内容 ******************************************************************

・このデータは東方Project「博麗霊夢」の「MikuMikuDance」用モデルデータです。
　”一応”環境マップを入れていますので最新版のＭＭＤをお使い下さい。

・MMD付属モデルと比較して目の可動域が狭いので、ミクのモーションを使用した場合
高確率で目が飛び出す、白目を剥く等の問題が出ます。
両目のボーンモーションの修正で直ります、覚えておいてください。

左目・右目・両目ボーンを全フレーム選択状態にして
フレーム位置角度補正を回転0.4〜0.6あたりで適応すると良いかもしれません。
（霊夢用に修正したストロボナイツのモーションを同梱しました）

・エッヂ太さは0.3辺りで使ってください。

・不具合＆要望があれば適当に連絡下さい、暇を見て対応します。
